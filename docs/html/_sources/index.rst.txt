Purpose
=======

This site hosts the Jupyter notebooks for the CBC Bioconductor Workshop 2.  The notebooks will be served on the web as static HTML pages, meaning that while they looks like live notebooks, they cannot be edited.  To obtain an editable copy of the notebook files, please click the respective 'Download Notebook' link below.  These 4 notebooks were created by Levi Waldron.
